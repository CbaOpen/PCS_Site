Les instruments de collecte

Vous trouverez ici l’information nécessaire à l’obtention de la nomenclature socioprofessionnelle selon le protocole rénové de codage :

  • la liste des libellés standardisés, qui peut être parcourue par le biais d’un tableur ou interrogée grâce à l’application d’auto-complétion utilisée pour la collecte) ;

  • le module des questions (ou variables « annexes ») nécessaires à la codification (statut, nombre de salariés et position professionnelle ; spécifiquement le type de contrat de travail pour les classes d’emploi), questions dont une formulation multi-mode est fournie.

Ces éléments peuvent être exportés aisément dans différents langages, afin de permettre leur utilisation dans une large gamme d’environnements de collecte.
